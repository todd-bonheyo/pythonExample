# pythonExample

ITS 211 — Python example repository

Overview
-	Small repository containing example code used in the ITS 211 course. The primary example is a simple `VendingMachine` class implemented in a single interactive script.

Files
- `11.3 - vending machines.py`: Interactive script demonstrating the `VendingMachine` class. The script prompts for current stock and an order quantity, then shows stock changes and refill behavior.

Requirements
- Python 3.8 or newer

Run
- From this folder run:

```bash
python3 "11.3 - vending machines.py"
```

Notes
- The script uses `input()` and is interactive — run it in a terminal.
- Save any edits before running so changes are picked up by Git.
